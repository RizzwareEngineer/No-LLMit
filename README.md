# No-LLMit

Spectate (or play against) SOTA LLMs in a No Limit Texas Hold'em cash game (or, soon, tournament)!

## Features
* [LLM Council (Andrej Karpathy)](https://github.com/karpathy/llm-council)
* [Tinker (Thinking Machine's Lab)](https://github.com/thinking-machines-lab/tinker-cookbook)


## Limitations
We estimate that it takes ~30 API requests (to HuggingFace Serverless API) from start-to-finish of a single hand. 
Unfortunately, I am limited to 20K API requests a day using my personal HuggingFace Pro account which roughly results to 667 hands a day.
